## Prerequisites

All tools listed below must be available for use!

- Create a GitHub account
- Git must be installed
- Terminal
- VS Code
- Live Server extension must be installed in VS Code
- Web Browser **(Chrome Recommended)**
- Internet Connection

## Step-by-step Guide

1. Open terminal on your computer.

2. Navigate to your preferred directory:
```bash
cd ~
```

3. Clone the repository:
```bash
git clone https://github.com/Douglasebhoman/douglasebhoman.github.io.git
```

4. Move into the project folder:
```bash
cd douglasebhoman.github.io
```

5. Open the project in VS Code:
```bash
code .
```
> **Note:** This requires VS Code to be added to PATH during installation. If it doesn't work, open VS Code manually and use **File → Open Folder**.

6. In the VS Code file explorer, click `index.html` to open it.

7. Right-click `index.html` and select **Open with Live Server**.

8. Your default browser will open the site at `http://127.0.0.1:5500`
